Styling methods

Inline styles

This word is styled using the standard inline HTML style attribute, e.g style="color: blue; font-weight: bold"

This word is also styled using the inline method however it uses Angular property bindings and the style value is sourced from a property belonging to the component class, e.g [style.color] = "darkRed"

External styles
