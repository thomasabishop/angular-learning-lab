Render if data is returned

This variation shows a useful application of conditional rendering: we make the rendering conditional on whether or not the data exists for a given array element. For instance, if we make a request to an API and one instance of the returned data is either undefined or null doesn't that element won't be rendered at all, rather than displaying undefined or broken data. I have added two non-compliant values: `null` and `undefined` to the data array to simulate this.
