Really basic observable

We are going to create a basic observable.

The template variable below stores a number value which an Observable increments by one every second. This subscription is destroyed when the component is derendered.

We have used the error() method on the Observable to emit an error if the value of the number is equal to 5. This is why the output stops at 4. We could also just call .complete() on the Observable. In both cases the subscription ends.
